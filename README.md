# 文件互传与分享系统

这是一个基于Node.js的文件互传与分享系统，支持手机和电脑之间像聊天一样互传文件，也可以暂存文件供他人下载。

## 功能特性

- 📤 文件上传：支持从本地选择文件上传
- 📥 文件下载：提供文件下载链接
- 💬 实时聊天：支持实时消息通信
- 📁 文件暂存：上传的文件会暂存在服务器上
- 🔗 分享功能：通过链接分享文件给他人

## 技术栈

- **后端**：Node.js + Express + Socket.io
- **前端**：HTML + CSS + JavaScript + EJS
- **文件处理**：Multer + fs-extra

## 安装步骤

1. **克隆项目**（如果从版本控制系统获取）
   ```bash
   git clone <repository-url>
   cd <project-directory>
   ```

2. **安装依赖**
   ```bash
   npm install
   ```

3. **启动服务器**
   ```bash
   npm start
   ```

4. **访问系统**
   打开浏览器，访问 `http://localhost:3000`

## 使用方法

### 上传文件
1. 在"上传文件"区域点击"选择文件"按钮
2. 选择要上传的文件
3. 点击"上传"按钮
4. 上传成功后，文件会显示在"最近上传的文件"列表中

### 下载文件
1. 在"最近上传的文件"列表中找到要下载的文件
2. 点击文件右侧的"下载"链接
3. 文件会自动开始下载

### 实时聊天
1. 在"聊天"区域的输入框中输入消息
2. 点击"发送"按钮或按Enter键
3. 消息会实时显示在聊天记录中，并同步给所有连接的用户

## 注意事项

- 上传的文件会存储在 `uploads` 目录中，请确保该目录有写入权限
- 本项目仅用于本地或局域网内使用，不建议直接部署到公网
- 上传的文件会暂存在服务器上，定期清理以节省空间
